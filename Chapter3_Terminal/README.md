## 1. *常用命令 Common commands*
|Windows|MacOS/Linux|説明 instruction|
|----|----|----|
|cd|cd|切換目錄|
|cd|pwd|取得目前所在位置|
|dir|ls|列出當前的檔案列表|
|mkdir|mkdir|建立新的目錄|
|無|touch|建立新的檔案|
|copy|cp|複製檔案|
|move|mv|移動檔案|
|del|rm|刪除檔案|
|cls|clear|清楚畫面上内容|
